你有没有想过，`bing.com`、`google.com`、`taobao.com` 这些网址是怎么来的？为什么你能直接在浏览器里输入这几个字母就能打开对应的网站？答案就是——**域名**。这篇文章专门会用最通俗的大白话，解释清楚域名到底是什么、能干什么、免费和付费域名具体怎么选，全程不涉及任何专业术语哦～

# 一、域名到底是啥？

## 1.1 用生活中的例子来解释

你可以把**域名**理解为网站的"门牌号"。

就好比你家有个地址："上海市静安区某某路1号"。别人只要知道这个地址，就能找到你家。

网站也一样。每台存放网页内容的服务器，都有一个数字地址（叫 **IP 地址**，比如 `202.102.134.68`）。但是这种数字太难记了，于是就有了域名，把数字翻译成好记的字母组合：

|真实门牌|类比到网站|
|---|---|
|北京市朝阳区某某路1号|`bing.com`|
|IP地址 = 服务器的数字身份|`202.102.134.68`|

你输入 `bing.com`，DNS 系统自动把它翻译成对应的 IP 地址，浏览器再找到那台服务器，网页就出现了。整个过程不到一秒钟。

## 1.2 域名的结构长啥样？

以 `www.baidu.com` 为例：

`www` . `baidu` . `com`

↑       ↑           ↑

子域     二级域     顶级域

- **.com** 叫**顶级域**（Top-Level Domain，TLD），类似于城市的"市辖区"分类。常见的顶级域有 `.com`（商业）、`.cn`（中国）、`.org`（组织）、`.net`（网络）等，全世界有几千种。

- **baidu** 叫**二级域**，是你花钱买下来的"名字"，全世界独一无二。

- **www** 叫**子域/三级域**，你可以随便设置，`blog.baidu.com`、`shop.baidu.com` 都行。

所以严格来说，你花钱买的是"baidu.com"这个二级域，www、blog 这些子域都是附赠的，随便你怎么用。

## 1.3 域名后缀的价格差距有多大？

同样是一个"名字"，不同后缀价格天差地别：

|后缀|价格区间|适合场景|
|---|---|---|
|`.com`|50~200元/年|正经商业网站，搜索引擎首选|
|`.cn`|20~80元/年|国内业务|
|`.xyz` / `.top`|1~10元/年|尝鲜、试水、个人项目|
|`.io`|200~800元/年|程序员/科技公司心头好|
|`.ai`|1000+元/年|AI 相关创业公司|

最离谱的是 `.com`，同样是两个字的名字，2024年有个 `sex.com` 域名卖了几亿美元——没错，就三个字母，价值几亿美元。所以好的 `.com` 域名，本身就是一种资产，当然这么好的域名是很难抢到的。

# 二、域名能干什么？

## 2.1 搭建网站

这是最常见的用途。买一个域名，配上一台服务器（或用 GitHub Pages、Vercel 等免费托管平台），放上代码和文件，就能拥有自己的网站。个人博客、作品集、小的工具站，都可以从这里直接开始。

## 2.2 绑定各类服务

除了搭建网站，域名还能绑定：

- **邮箱**：用你自己的域名开邮箱，最简单的还是用大善人 Cloudflare 开域名邮箱，但目前只能收件，不能发件。比如我自己注册的域名邮箱 `i@fivor.cc`，可比 `123456789@qq.com` 显得专业多了

- **图床**：把自己的图片存起来，用域名访问，Github上有很多现成的图床项目，比如 [CloudFlare-ImgBed](https://github.com/MarSeventh/CloudFlare-ImgBed)。

- **内网穿透**：人在外面，也能通过域名访问家里的 NAS 或软路由，或者电脑上的 Docker 项目。比如我现在就是在家里电脑上装了 Docker，然后通过 Cloudflare Tunnel 绑定自己的域名，就能在外面访问我的电脑上的服务。

- **API 服务**：给自己的接口起个好记的名字，比如 `api.fivor.cc`，就能在浏览器里直接访问。

## 2.3 收藏/投资

有些人专门囤积短域名等升值。这个就不在本文讨论范围，但知道一下就好——域名投资是个专业活，小白别轻易入坑。

# 三、免费域名：能白嫖吗？

先说结论：**能，但要分清楚是"永久免费"还是"试用免费"**。很多平台打着"免费"的旗号，其实只是首年免费，第二年就开始收费，甚至还不便宜，这种小白千万慎入。

## 3.1 免费域名的一些坑，你得先知道

**免费的代价是什么？**

1. **平台说关就关**：2023年大名鼎鼎的 **Freenom**（提供 `.tk` `.ml` `.ga` `.cf` `.gq` 免费顶级域名）因为被 Meta 起诉滥用，**直接停止新用户注册**。无数人用了几年的域名一夜之间全没了。所以免费域名**永远存在平台跑路或关停的风险**。

2. **不稳定**：免费域名解析可能随时出问题，网站打不开是家常便饭。

3. **隐私公开**：有些免费平台不提供 WHOIS 隐私保护，你的注册信息（姓名、邮箱）会被全世界看到。

4. **限制多**：免费域名很多都不支持修改 NameServer 记录托管到 Cloudflare，不能设置 SPF/DKIM 邮件验证，功能阉割严重。

## 3.2 2026年实测可用的免费域名平台

下面列出目前还能用、且有一定口碑的平台，按推荐程度排序：

### 🥇 DNSHE（强烈推荐）

这是目前我用过最好用的免费域名平台，量大管饱，比如我自己就已经白嫖了十几个免费域名。

- **可用后缀**：`de5.net`、`cc.cd`、`ccwu.cc`、`us.ci`、`cn.mt`、`bbroot.com` 等

- **支持 Cloudflare**：部分后缀可以托管到 Cloudflare，享受 CDN 加速

- **注册额度**：新用户 3 个名额，通过邀请最多加 2 个，共 5 个

- **注册地址**：[DNSHE](https://my.dnshe.com)

**小技巧**：DNSHE 最近还新增了 `.cn.mt` 和 `bbroot.com` 后缀，但需要专用邀请码才能注册，暂时不支持 Cloudflare。

### 🥈 EU.CC（GName 官方活动）

这是新加坡域名注册商 GName 搞的促销活动，不是真永久免费，但**首年只要 1.88 美元（约 14 元人民币）**，比很多付费域名都便宜。

- **可用后缀**：`.eu.cc`

- **特点**：邮箱验证 + 谷歌身份验证器 MFA，不需要手机号

- **注册额度**：一个账号 3 个

- **重要**：到期前 90 天要去活动页手动点"免费续费"，自动续费是不免费的

- **注册地址**：[EU.CC](https://gname.com/tld-eu-cc.html)

### 🥉 EU.ORG（老牌永久免费，但门槛高）

从 1996 年开始运营的元老级免费域名，真正的永久免费、不用续期，**可以绑 Cloudflare**。

- **可用后缀**：`.eu.org`（但跟欧洲没关系）

- **问题**：人工审核，排队时间可能很长，短则几周，长则几个月到半年

- **适合人群**：有耐心、真正想长期稳定使用的用户

- **注册地址**：[EU.ORG](https://eu.org/)

## 3.3 免费域名最佳实践

**免费域名最佳实践：**

1. **不要把所有鸡蛋放一个篮子**：多注册几个不同平台的域名，主站用付费域名，备用站用免费域名

2. **重要项目不要用免费域名**：你的个人品牌网站、作品集、简历站，强烈建议花十几块钱买个正经 `.com` 或 `.cn`

3. **定期检查续期状态**：很多免费域名每年要手动续期，忘了就没了

4. **优先选能绑 Cloudflare 的**：Cloudflare 的免费 CDN 能显著提升访问速度和稳定性，推荐小白直接选能托管 Cloudflare 的免费域名，能省下很多事

# 四、国内 vs 国外：注册域名有什么区别？

这是很多小白最困惑的问题。答案很简单：**都能用，但体验完全不同。**

## 4.1 国内注册（阿里云 / 腾讯云）

**优点**

- 中文界面，有客服，沟通无障碍

- 支付宝/微信直接付款，不用信用卡

- 域名备案（ICP 备案）后，可以绑定国内服务器，速度快，搜索引擎（百度）收录好

- 域名解析速度快，稳定性好

**缺点**

- 必须完成 ICP 备案才能绑定国内服务器，备案流程繁琐（7~30 天）

- WHOIS 信息强制实名，隐私保护弱

- 域名转出相对麻烦，部分平台有转出限制

- 价格比国外略贵（但常有活动）

**什么是 ICP 备案？** 简单说就是向工信部登记你的网站，说明"这个域名是谁的，用来干什么"。不备案的话，域名绑定国内服务器会被墙访问不了。但如果你用国外服务器（美国/新加坡等），就不需要备案。

## 4.2 国外注册（Namecheap / GoDaddy / Spaceship / NameSilo）

**优点**

- 价格便宜，特别是首年优惠（比如 `.com` 首年 $1）

- WHOIS 隐私保护（很多平台永久免费）

- 域名转出自由，没有太多限制

- 不需要备案，买了就能用

**缺点**

- 英文界面（但现在翻译软件很方便）

- 需要信用卡或 PayPal 付款

- 解析速度在国内可能稍慢（但绑了 Cloudflare 就解决了）

- 没有中文客服，出问题沟通成本高

- 域名如果涉及敏感内容，有被起诉/没收的风险（罕见但确实存在）

## 4.3 到底选国内还是国外？

|你的情况|建议|
|---|---|
|网站面向国内用户，要用国内服务器|国内注册 + 备案|
|只是个人学习/测试/内网穿透|国外注册，不备案|
|想做 SEO（百度搜索排名）|国内注册 + 备案|
|主要面向海外用户|国外注册|
|预算有限，想省钱|国外注册（首年优惠多）|
|想要 `.cn` 中文域名|必须国内注册|

# 五、付费域名去哪儿买？

如果你决定花钱买个正经域名，下面这些平台是目前最主流、最靠谱的选择。

## 5.1 国外平台（推荐）

### 🚀 Spaceship（性价比之王）

最近几年崛起的新秀，界面清爽，价格透明，支持支付宝。

- `.com` 首年特惠：使用优惠码 `COMPROS`，约 **¥20**（约 $2.8）

- **数字 .xyz 域名**：6位纯数字（如 `123456.xyz`），仅 **¥4.65/年**，10年打包价 ¥46.5，是目前长期持有最低成本的方案之一

- 支持支付宝

- 缺点：`.com` 首年优惠后，次年续费恢复到 ¥70/年左右

**省钱技巧**：`.xyz` 数字域名续费也是 ¥4.65/年，如果你想注册一个长期持有的便宜域名，数字 `.xyz` 是首选。

### 🏆 NameSilo（老牌稳定）

老牌域名注册商，以"没有隐性消费"著称，价格透明，口碑极好。

- `.top`、`.lol`、`.cyou` 等后缀，首年使用优惠码 `Appstore`，**仅需 $1 左右**

- **永久免费 WHOIS 隐私保护**（很多平台这项要收费）

- 支持支付宝

- `.com` 续费约 ¥70/年，价格稳定

**适合人群**：想买 `.com` 但想首年省钱，同时想要稳定续费价格的人。

### 🔥 Google Workspace（土耳其区 .com 奇迹）

这是一个野路子，但我前不久亲测还能用。通过 Google Workspace 土耳其区通道注册 `.com` 域名，折合人民币仅需 **¥14/年**，比 Spaceship 还便宜。

- 需要准备：土耳其节点 + 支持外币的信用卡

- **重要**：注册并购买域名后，务必去后台取消 "Google Workspace 商业标准版" 订阅，否则每个月会被扣 Workspace 套件费（很贵）

**风险提示**：这个方法依赖 Google 的区域定价政策，Google 可能随时调整，建议趁能注册时一次多买几年。

## 5.2 国内平台

### ☁️ 阿里云万网

国内最老牌、最稳定的域名注册平台。

- `.com` 首年约 ¥55~80（常有活动）

- `.cn` 约 ¥25~35/年

- 优点：备案方便，客服响应快，和阿里云服务器绑定丝滑

- 缺点：续费相对较贵，WHOIS 隐私保护要单独付费

### 🐧 腾讯云 DNSPod

依托 DNSPod 的域名解析优势，腾讯云的域名注册体验也很好。

- `.com` 首年活动价 ¥33（限时）

- 和腾讯云服务器、CDN 绑定非常方便

- 缺点：和阿里云类似，续费价格比国外高

## 5.3 付费域名选购建议总结

|**场景**|**推荐选择**|**理由**|
|---|---|---|
|只想花最少钱拿个 `.com`|Spaceship (¥20) 或 Google Workspace (¥14)|首年价格最低|
|长期持有，低成本|数字 `.xyz`（¥4.65/年）|续费也便宜|
|想要稳定不踩坑|NameSilo ($1首年，透明续费)|没有套路|
|国内业务，需要备案|阿里云 / 腾讯云|备案一条龙|
|面向程序员/技术圈|`.io` 域名（NameSilo，约 ¥200/年）|圈子里的"显档"后缀|

# 六、域名注册后的注意事项

## 6.1 域名解析是什么？

买完域名只是第一步，你还需要告诉域名"我的服务器在哪"，这个过程叫**域名解析**（DNS 配置）。

简单来说，就是添加 DNS 记录：

|记录类型|作用|举例|
|---|---|---|
|**A 记录**|把域名指向一个 IP 地址|`blog.myname.com` → `192.168.1.1`|
|**CNAME**|把域名指向另一个域名|`www.myname.com` → `myname.github.io`|
|**MX 记录**|指定邮件服务器|`myname.com` → `mx.mail.com`|
|**TXT 记录**|用于验证、SPF 邮件反垃圾等|Google Search Console 验证|

最简单的方式是用 **Cloudflare** 来托管你的域名 DNS——免费、稳定、速度快，而且是中文界面。

## 6.2 隐私保护很重要

域名注册信息（姓名、电话、邮箱）默认会记录在 WHOIS 数据库里，全世界都能查到。不想被骚扰？

- **国外平台**：NameSilo 等平台永久免费提供隐私保护

- **国内平台**：阿里云/腾讯云有"域名隐私保护"服务，通常收费（¥10~30/年）

- **免费方案**：用 Cloudflare 托管 DNS 后，WHOIS 显示的是 Cloudflare 的信息，你的真实信息被隐藏

## 6.3 域名到期续费

这是最容易踩坑的地方：

1. **提前续费**：域名到期前 30 天内才能续费，过期后有 30~45 天赎回期（赎回费很贵），再之后就被释放给其他人注册了

2. **开启自动续费**：省心，不会忘记

3. **域名抢注**：好域名到期不续，很快就会被人抢走，价格翻几十倍

4. **到期监控**：后续我会单独出一个教程，用 Github 上的开源项目 [domain-check](https://github.com/yutian81/domain-check)，5分钟搭建一个域名到期监控网站，域名在到期前会在 Telegram 发消息提醒自己续费

## 6.4 不要把所有域名放在同一个平台

就像不要把所有钱存同一家银行一样，建议把重要域名的 DNS 和注册商分开管理，降低"平台出故障全灭"的风险。

# 七、总结

**想白嫖？**

- 随便玩玩，不重要 → DNSHE / ZoneABC / INDEVS.in（免费，能绑 Cloudflare）

- 想要长期稳定 → EU.ORG / NIC.UA pp.ua（人工审核，但极稳定）

- 只是DDNS → Dynu / NDJP.NET（后缀多，专门做动态域名）

**想花钱买？**

- 预算 ¥10~20/年 → 数字 `.xyz`（Spaceship，¥4.65/年）

- 想拿 `.com`，但想省钱 → Spaceship 首年 ¥20 / Google Workspace ¥14

- 想要稳定不踩坑 → NameSilo（$1首年，透明续费）

- 国内业务，要备案 → 阿里云 / 腾讯云

**国内外怎么选？**

- 国内服务器 + 百度SEO → 国内注册 + 备案

- 国外服务器 / 个人项目 → 国外注册，不需要备案

- 想要隐私保护 → 国外平台（NameSilo等）

# 最后说几句

域名这东西，买对不买贵。对于新手来说，**先花几块钱买个 `.xyz` 或 `.top` 试试水**，搞清楚整个流程后，再决定要不要投资更好的 `.com`。

免费域名诚然诱人，但请始终记住那句话：**免费的东西，往往在最贵的时候让你付出代价**——当你辛辛苦苦搭了几个月的网站，域名说没就没了，那种滋味只有经历过的人才懂。

所以，**重要的事情用付费域名，不重要的折腾用免费域名**，这才是正确的打开方式。

祝大家都能拿到自己心仪的域名，网站越做越好 🚀
